# Design Pattern

A collection of design patterns written in java. See "Gang of Four Design Patterns".

## Behavioural Pattern
Contains the following well known pattern:
* chain of responsibility
* command
* interpreter
* iterator
* mediator
* memento
* observer
* state
* strategy
* template
* visitor

## Creational Pattern
Contains the following well known pattern:
* factories
* builder
* object pool
* prototype
* singleton

## Structural Pattern
Contains the following well known pattern:
* adapter
* bridge
* composite
* decorator
* facade
* flyweight
* lazy evaluation
* proxy
