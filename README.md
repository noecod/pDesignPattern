# Design Pattern
A collection of design patterns written in java. See "Gang of Four Design Patterns".

## Behavioural Patterns
Contains the following well known patterns:
* chain of responsibility
* command
* interpreter
* iterator
* mediator
* memento
* observer
* state
* strategy
* template
* visitor

## Creational Patterns
Contains the following well known patterns:
* factories
* builder
* object pool
* prototype
* singleton

## Structural Patterns
Contains the following well known patterns:
* adapter
* bridge
* composite
* decorator
* facade
* flyweight
* lazy evaluation
* proxy
